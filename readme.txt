== Installation ==
1. Upload `best-slider-testimonial` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Copy and paste <strong>if(function_exists('wpbt_custom_shortcode')) { wpbt_testimonial_loop(); }</strong> in template code or  <strong>[BESTSLIDERTESTIMONIAL]</strong> in the post/page" to display testimonial.
